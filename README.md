## Format Dokumen (sesuai ketentuan lomba)
- A4, margin 4-4-3-3 cm
- Times New Roman 12 (Latin), Traditional Arabic 16 (Arab)
- Spasi 1.5
- Struktur: Pendahuluan → Landasan Teori → Tawaran Gagasan → Penutup
- Minimal 1 ayat Al-Qur'an utama + tafsir
